# ROBO-DOC
Healthcare chatbot to predict Diseases based on patient symptoms.

# How to run:
## create a venv 
virtualenv venv 

## activate it and install reqs
- source venv/bin/activate
- pip install -r requirements.txt 
- python -m spacy download en_core_web_sm

## run app file
python app.py
